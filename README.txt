Pour pouvoir lancer le programme, il suffit de lancer le fichier "cartes.exe" contenu dans le dossier "bin"(NE PAS OUBLIER DE METTRE "cartes.txt" DANS LE MEME DOSSIER).
Ou bien d'ouvrir "cartes.cbp" avec Code::Blocks et de lancer le programme.
Vous pourrez soit jouer contre une IA ou un humain. Pour cela il faudra entrer soit "H" pour humain soit "I" pour IA.
Il faudra ensuite entrer le nom des deux joueurs si c'est pour "H" ou un seul si c'est pour "I".
Pour ce faire, entrez le nom d'un joueur, appuyez sur Entrée, entrez le nom d'un autre joueur si c'est pour "I", reappuyez sur Entrée.
Chaque joueur pourra décider de sa propre stratégie pour sa pile de cartes, il faudra entrer un nombre entre 1 et 6 pour utiliser la stratégie souhaitée.
(NB : Si vous souhaitez utiliser la stratégie 6, il faudra entrer le nom de 20 cartes, piochées dans la réserve !
Si une carte est sélectionnée il sera affiché "Carte Selectionnee", dans le cas contraire (un nom mal écrit), le programme n'affichera rien.
/!\ Ne pas utiliser 2 fois la même carte)
Le combat se déroule automatiquement ensuite entre les deux joueurs, et le joueur gagnant est affiché avec son prestige à la fin !
Pour finir, le jeu demander au joueur si il veut rejouer ou non en entrant "o" ou "n".
